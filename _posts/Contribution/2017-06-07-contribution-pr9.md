## 충돌 수정

수정 내역을 받아오지 않고 번역을 진행하는 탓에 충돌을 경험하였다. 

충돌 발생시 깃허브에서 직접 충돌을 수정해 merge할 수 있다는 것을 알았다.

수정 내역을 받아올경우 
```
git pull
```
명령어를 이용한다.
